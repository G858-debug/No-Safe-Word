# Writing Guide

## Story Structure

Stories in No Safe Word follow a consistent structure optimized for both Facebook posts and website display.

### Facebook Content

- **Length**: 500-800 words per post
- **Images**: 2-3 images per post (SFW only)
- **Engagement**: Cliffhanger endings to encourage engagement
- **Metadata**: Include hashtags and teaser text

### Website Content

- **Length**: Full narrative with explicit scenes
- **Images**: Additional NSFW images paired with Facebook images
- **Inline Images**: Website-only images embedded inline
- **Formatting**: Richer formatting (bold, italic, quotes)

## Character Descriptions

When creating characters for image generation, include:

- **Physical Attributes**: Hair, eyes, body type
- **Clothing**: Style and specific garments
- **Expressions**: Facial expressions and mood
- **Distinguishing Features**: Unique characteristics

## Image Prompts

### Facebook SFW Images

- Suggestive but not explicit
- Focus on intimacy, tension, emotion
- Avoid nudity or explicit acts
- Professional quality, artistic composition

### Website NSFW Images

- Can be explicit
- Paired with corresponding Facebook images
- Follow platform guidelines
- Maintain artistic quality

## Best Practices

1. **Consistency**: Keep character descriptions consistent across parts
2. **Pacing**: Build tension gradually
3. **Cliffhangers**: End each part with anticipation
4. **Diversity**: Represent South African diversity
5. **Language**: Use South African English and local context
